Style Guide Template

A template you can use to start building out your own style guides with.

Default Button Style

CSS selector:

'a.button'

Sample code:

<a href="#" class="button">DEFAULT BUTTON</a>

Rendered element:

DEFAULT BUTTON DEFAULT BUTTON

Default Form Style

CSS selector:

'label.form'

'input.form'

'textarea.form'

'input.form-submit'

Sample code:

<div class="input-wrapper"><label class="form" for="form-name">Default Form:</label><input class="form" type="text" id="form-name" placeholder="Default Form"></div> <div class="input-wrapper"><textarea class="form" id="form-comment" placeholder="Default Form">Default Form</textarea></div><input class="form-submit" type="submit" value="DEFAULT FORM">

Default Textual Style

CSS selector:

'h3.title'

'p.textual'

'a.important'

Sample code:

<h2 class="title">Default Textual</h2><p class="textual">Default Textual, Etiam sit amet sem vehicula, efficitur eros faucibus, dignissim odio. <a class="important" href="#">Duis diam sem, malesuada</a> non augue quis, viverra lacinia turpis. Morbi in vehicula eros, quis vestibulum libero.</p>
